# data-analysis.md — 투자 데이터 분석 규칙

## §1. 데이터 전처리 규칙

### 1.1 입력 데이터 형식
- **필수 필드**: date, close (종가)
- **선택 필드**: open, high, low, volume, ticker, sector
- date는 ISO 8601 형식 또는 YYYY-MM-DD로 파싱

### 1.2 결측치 처리
| 상황 | 처리 |
|------|------|
| 종가 결측 (1~2일) | 전일 종가로 forward fill |
| 종가 결측 (3일 이상 연속) | 해당 구간 제외, UI에 경고 표시 |
| 거래량 결측 | 0으로 대체 |
| 전체 데이터가 30일 미만 | 분석 불가 경고 표시 |

### 1.3 수익률 계산
- **일간 수익률**: `(close_t - close_{t-1}) / close_{t-1}`
- **누적 수익률**: `(close_t / close_0) - 1`
- **연환산 수익률**: `(1 + 총수익률) ^ (252 / 거래일수) - 1`

## §2. 투자 지표 계산 규칙

### 2.1 핵심 지표
| 지표 | 계산법 | 용도 |
|------|--------|------|
| 수익률 (Return) | 기간별 누적 수익률 | 성과 비교 |
| 변동성 (Volatility) | 일간 수익률의 표준편차 × √252 | 위험 측정 |
| 샤프 비율 (Sharpe) | (연환산수익률 - 무위험수익률) / 변동성 | 위험 대비 수익 |
| 최대 낙폭 (MDD) | 고점 대비 최대 하락률 | 최악 시나리오 |
| 상관계수 | 피어슨 상관계수 (일간 수익률 기준) | 분산 투자 효과 |
| 베타 (Beta) | ETF 수익률 vs 벤치마크 회귀 기울기 | 시장 민감도 |

### 2.2 무위험 수익률
- 기본값: 연 4.0% (미국 T-Bill 기준)
- 설정 가능하도록 파라미터화

### 2.3 기간 필터 규칙
| 라벨 | 기간 |
|------|------|
| 1M | 최근 1개월 (21 거래일) |
| 3M | 최근 3개월 (63 거래일) |
| 6M | 최근 6개월 (126 거래일) |
| 1Y | 최근 1년 (252 거래일) |
| YTD | 연초 ~ 현재 |
| ALL | 전체 기간 |

## §3. 포트폴리오 분석 규칙

### 3.1 포트폴리오 수익률
- 가중 평균 수익률: `Σ(weight_i × return_i)`

### 3.2 포트폴리오 변동성
- `√(W^T × Σ × W)` (W: 비중 벡터, Σ: 공분산 행렬)

### 3.3 섹터 비중 분석
- 각 ETF의 섹터 태그 기반으로 섹터별 비중 합산
- 특정 섹터 비중 > 40% → 집중 리스크 경고

### 3.4 리밸런싱 시뮬레이션
- 입력: 목표 비중 배분
- 주기: 월간 / 분기 / 반기 / 연간
- 비교: 리밸런싱 vs 바이앤홀드 누적 수익률

## §4. ETF vs 개별종목 직접투자 비교 규칙

동일 섹터에 투자할 때, ETF를 사는 것과 개별 종목을 직접 골라 사는 것의 차이를 분석한다.

### 4.1 비교 대상 구성
- **ETF 포트폴리오**: 섹터 ETF 1종목 (예: XLK)
- **직접투자 포트폴리오**: 해당 섹터 상위 N개 종목 동일 비중 (Equal Weight)
  - N은 사용자 선택 가능 (기본값: 상위 5개)
  - 상위 기준: 시가총액 순

### 4.2 비교 시뮬레이션 조건
| 항목 | ETF | 직접투자 |
|------|-----|---------|
| 초기 투자금 | 동일 (기본 $10,000) | 동일 |
| 보수율 | ETF 운용보수 (연간, 일할 차감) | 0% |
| 거래비용 | 매수 1회 | 매수 N회 (종목 수 × 거래 수수료) |
| 리밸런싱 비용 | 없음 (ETF 내부 처리) | 리밸런싱 시 N회 거래 발생 |
| 거래 수수료 | 기본 0.1% (설정 가능) | 기본 0.1% (설정 가능) |

### 4.3 비교 지표
| 지표 | 설명 |
|------|------|
| 누적 수익률 차이 | ETF 누적수익률 - 직접투자 누적수익률 |
| 총 비용 비교 | ETF 보수 누적 vs 직접투자 거래비용 누적 |
| 변동성 차이 | ETF는 분산 내장, 직접투자는 N종목 한정 분산 |
| 추적 오차 (Tracking Diff) | 직접투자 포트폴리오가 ETF 대비 얼마나 벗어나는지 |
| 최대 낙폭 비교 | 하방 리스크 차이 |
| 샤프 비율 비교 | 위험 대비 수익 효율 |

### 4.4 시나리오 분석
사용자가 조건을 변경하며 비교할 수 있다:
- **종목 수 변경**: Top 3 / 5 / 10 / 20
- **비중 방식**: 동일 비중 vs 시가총액 비중
- **리밸런싱 주기**: 없음 / 월간 / 분기
- **투자 기간**: 1Y / 3Y / 5Y
