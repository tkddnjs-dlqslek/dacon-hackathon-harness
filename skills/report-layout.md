## §2. 섹션 렌더링 규칙

### 2.1 섹션 구성 요소
모든 섹션은 다음 구조를 따른다:
```
[섹션 제목] + [설명 텍스트(선택)] + [차트/테이블] + [인사이트(해당 시)]
```

### 2.2 빈 데이터 처리
- 데이터 없음 → 해당 섹션 숨기지 않고, "데이터 없음" placeholder 표시
- 분석 불가 (데이터 부족) → 회색 카드 + 필요 조건 안내

### 2.3 로딩 상태
- 각 섹션 독립 로딩 (Skeleton UI)
- 전체 페이지 blocking 금지

## §3. 네비게이션 규칙
- 상단 Nav: 대시보드 / 섹터 / 포트폴리오 / ETF vs 직접투자
- 섹터 목록: 드롭다운 또는 탭
- 차트 클릭 → 상세 페이지 이동
- URL로 상태 유지 (기간 필터 등 query param)
