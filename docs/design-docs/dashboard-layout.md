
## 인터랙션 흐름

### 기간 필터 (전역)
```
사용자: [1M] 클릭
  → URL query: ?period=1m
  → data-loader: 해당 기간 데이터 필터
  → analysis-engine: 지표 재계산
  → chart: 리렌더
  → insight-generator: 인사이트 재생성
```

### 포트폴리오 비중 변경
```
사용자: XLK 슬라이더 25% → 30%
  → 나머지 비중 자동 비례 조정 (합계 100% 유지)
  → 백테스트 재실행
  → KPI + 차트 + 인사이트 동시 갱신
```

### 섹터 드릴다운
```
사용자: 히트맵 or 바차트에서 "Technology" 클릭
  → /sector/technology 이동
  → 해당 섹터 ETF(XLK) + 상위 5 종목 데이터 로드
```

### ETF vs 직접투자 설정 변경
```
사용자: 종목 수 5 → 10 변경
  → 해당 섹터 상위 10 종목 데이터 로드
  → 시뮬레이션 재실행 (비중 재계산)
  → 전체 비교 지표 + 차트 + 인사이트 갱신
```
